# 抗体 Hotspot 智能预测平台完整需求文档 (PRD)

## 1. 产品定义

本工具是一个集序列扫描与三维结构校验于一体的抗体开发性评估系统。通过结合序列基序（Motif）扫描与结构空间可及性（RSA）分析，辅助研究员识别抗体药物在生产与储存中的理化风险点。

## 2. 核心算法与业务规则

### 2.1 序列分析与区域识别

- **标准化编号**：集成 `ANARCI` 库，支持 Kabat（默认）、Chothia 方案。
- **区域划分**：自动识别 CDR1/2/3、Framework (FR) 及 Fc 区。
- **可视化染色规范**：
  - CDR1/2/3：浅蓝、中蓝、深蓝。
  - Fc 区：浅黄色。
  - FR 区：透明灰色 (0.3 opacity)。

### 2.3 结构校准逻辑 (RSA 过滤)

- **计算逻辑**：调用 `FreeSASA` 计算相对表面积 (RSA)。
- **过滤规则**：
  - **深埋 (Buried)**：若 $RSA < 5$，即便命中基序，也从风险列表剔除，移动至“结构屏蔽清单”。
  - **暴露 (Exposed)**：若 $RSA > 20$，维持基序原始风险等级。
  - **关键区强化**：若风险点位于 CDR 区且 $RSA > 20$，等级提升为“Critical”。

## 3. 功能模块与交互规范

### 3.1 3D 可视化 (基于 Mol*)

- **双向联动**：点击序列位点，Mol* 自动 Focus 并以 Ball & Stick 模式展示残基；点击 3D 原子，序列视图同步高亮。
- **风险标记**：在 3D 结构上用红（高）、橙（中）、黄（低）球体标注风险点。

### 3.2 输入

- **序列输入：支持用户直接在输入框内输入序列。**
- **结构输入：支持用户上传蛋白pdb格式的文件。**

### 3.3 结果输出

- **导出功能**：支持 CSV 导出，包含：位点、Kabat 编号、风险类型、RSA 分值、所属区域。
- **自定义配置**：支持用户手动输入正则基序并保存。
- 点击“开始扫描”，左侧显示 Hotspot 列表，右侧渲染 3D 结构

## 4. 技术栈要求

- **前端**：React, Vite, Tailwind CSS, Mol*。
- **后端**：FastAPI (Python), Biopython, ANARCI, FreeSASA, AlphaFold API。
